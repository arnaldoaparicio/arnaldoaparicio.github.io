<p>Book Club</p>
<p>[Group Project]</p>
<p>A web app where users can search for books, create a club, invite other users, and start discussions within the club about a specific book.</p>
<p>Focus Areas: Build a separate frontend and backend, agile project workflow, implement authentication via OAuth</p>
<p>Timeline: 2 weeks</p>
<p>Technologies: Ruby, Rails, REST API, Heroku, HTML,Google OAuth2, Google Books API, PostgreSQL, NYTimes Books API </p>
