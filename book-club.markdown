# Book Club

[Group Project]

_A web app where users can search for books, create a club, invite other users, and start discussions within the club about a specific book._

Focus Areas: _Build a separate frontend and backend, agile project workflow, implement authentication via OAuth_

Timeline: _2 weeks_

Technologies: _Ruby, Rails, REST API, Heroku, HTML,Google OAuth2, Google Books API, PostgreSQL, NYTimes Books API_
