MUGEN DB

[Solo Project]

A web app that hosts various links to resources for the game engine called MUGEN.

Focus Areas: Implement Amazon S3, upload images to Amazon S3 bucket, create user based submissions, display uploaded images

Technologies: Ruby, Rails, Amazon S3, HTML, PostgreSQL, fly.io

Context: This project was created to implement image uploading with Amazon S3 to an app that consists of a Rails API backend and a Rails frontend. Currently, only a registered admin is allowed to publish content.
